PCB - or Process Control Block

Information management. Records and allows access to information about running processes. 

#+ATTR_LATEX: :caption \bicaption{---} :float multicolumn

Information we can get from a PCB:
- Process State
- Process Number
- Program Counter
- Memory Limits
- List of open files

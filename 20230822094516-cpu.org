#+title: CPU
#+filetags: :OS:

CPU - Central Processing Unit
Executes basic logical instructions

Can only perform basic tasks:
- Addition/Subtraction
- Multiplication/Division
- Shift operation
- Jump operation (JMP)
- Compare (CMP) - produces = and not = (!=)
- Combine CMP and JMP for:
  - JLT (jump less than)
  - JGT (jump greater than)





Connected to [[id:8fa836e1-a0ad-4b48-b2f3-bc61e2c91893][RAM]] via a Memory Bus.
Higher level instructions must be [[id:8b260d84-a4a5-43eb-928a-7661d2035013][Compiled]] in order for the CPU to properly execute.

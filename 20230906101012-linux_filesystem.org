#+title: Linux Filesystem
#+filetags: :OS:

Starts at the root directory ("/")
Subdirectories of the root include:
- /bin - *Essential* command binaries
- /boot - Static files, used by the bootloader
- /dev - Device Files
- /etc - Host specific system configuration
- /home - User home directories
- /lib - Shared Libraries
- /media - Removeable Media
- /mnt - Mounted Filesystem
- /opt - Add-on application software packages
- /sbin - System Binaries
- /srv - Date for service from system
- /tmp - Temp
- /usr - User utilities and applications
- /prov - process info

  There are a variety of different binary directories:
  - /bin
  - /sbin
  - /usr/bin
  - /usr/sbin
  - /usr/local/bin
  - /usr/local/sbin
  - /opt

Files in linux can have [[id:e90743ed-966b-4e58-8a07-f750b469339e][File Signatures]]. These are not the same as [[id:7d6a011b-a895-450e-b1e1-21d272b34d4b][File Extensions]].
